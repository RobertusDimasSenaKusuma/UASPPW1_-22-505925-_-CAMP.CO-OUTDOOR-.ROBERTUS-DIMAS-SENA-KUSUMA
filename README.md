# KRITERIA PENILAIAN

 Desain rapi mengikuti kaidah atau prinsip desain
Berikut adalah penjelasan untuk setiap bagian kode yang memenuhi persyaratan dasar desain :

1.Konsistensi
Konsistensi dalam desain dicapai dengan menggunakan kelas dan struktur yang serupa di seluruh kode CSS. Misalnya, penggunaan .home-bg .home .content dan .home-category .box-container .box menunjukkan konsistensi dalam struktur selector.

2.Kontras
Kontras dapat dilihat dalam penggunaan warna dan perbedaan ukuran teks. Misalnya, pada .home-bg .home .content span, warna teks yang cerah (--WHITE) digunakan untuk menyoroti elemen tertentu dan menciptakan kontras dengan latar belakangnya. Hal yang sama berlaku untuk .home-bg .home .content h3 dengan menggunakan warna teks TERANG (--WHITE).

3.Repetisi
Repetisi dapat dilihat dalam penggunaan selector dan properti yang diulang. Misalnya, dalam .home-category .box-container .box dan .products .box-container .box, properti seperti padding, text-align, border, background-color, box-shadow, dan border-radius digunakan secara berulang.

4.Kesatuan
Kesatuan dicapai melalui penggunaan kelas dan gaya yang serupa di seluruh kode CSS. Misalnya, .box-container digunakan dalam beberapa kelas seperti .home-category .box-container, .products .box-container, .reviews .box-container, dll. Ini menciptakan konsistensi visual dan kesatuan antara elemen-elemen yang terkait.

//

Kode yang diberikan adalah aturan media query CSS yang menargetkan perangkat dengan lebar layar tertentu. Setiap aturan media query memiliki aturan CSS yang berbeda untuk diterapkan pada perangkat dengan lebar layar yang sesuai.

Berikut adalah penjelasan dari setiap media query:

1. `@media (max-width: 991px)`: Ini adalah media query untuk perangkat dengan lebar layar maksimum 991 piksel. Di dalamnya, terdapat aturan CSS untuk mengubah ukuran font pada elemen HTML menjadi 55% dari ukuran font asli.

2. `@media (max-width: 768px)`: Media query ini menargetkan perangkat dengan lebar layar maksimum 768 piksel. Di dalamnya, terdapat beberapa aturan CSS:

   - `#menu-btn`: Aturan ini mengubah tampilan elemen dengan ID "menu-btn" menjadi `display: inline-block`, yang berarti elemen tersebut akan ditampilkan sebagai blok yang inline.

   - `.header .flex .navbar`: Aturan ini mengubah tampilan elemen dengan class "navbar" yang berada di dalam elemen dengan class "flex" yang berada di dalam elemen dengan class "header". Aturan ini mengatur posisi elemen menjadi absolut dan menentukan tampilan klip jalan yang berbeda saat elemen tersebut dalam keadaan aktif atau tidak aktif.

   - `.header .flex .navbar a`: Aturan ini mengubah tampilan link di dalam elemen dengan class "navbar". Aturan ini mengatur tampilan link menjadi blok dan memberikan margin sebesar 2rem.

   - `.home-bg .home .slide .content`: Aturan ini mengubah tampilan konten dalam elemen dengan class "content" yang berada di dalam elemen dengan class "slide" yang berada di dalam elemen dengan class "home-bg". Aturan ini mengatur tampilan teks menjadi tengah (text-align: center).

   - `.home-bg .home .slide .content h3`: Aturan ini mengubah tampilan elemen h3 di dalam elemen dengan class "content" yang berada di dalam elemen dengan class "slide" yang berada di dalam elemen dengan class "home-bg". Aturan ini mengatur ukuran font elemen h3 menjadi 3rem.

3. `@media (max-width: 450px)`: Media query ini menargetkan perangkat dengan lebar layar maksimum 450 piksel. Di dalamnya, terdapat beberapa aturan CSS:

   - `html`: Aturan ini mengubah ukuran font pada elemen HTML menjadi 50% dari ukuran font asli.

   - `.heading`: Aturan ini mengubah ukuran font pada elemen dengan class "heading" menjadi 3rem.

   - `.flex-btn`: Aturan ini mengubah tampilan elemen dengan class "flex-btn" untuk mengatur tata letaknya menjadi kolom dan menghapus jarak antar elemen anaknya (gap: 0).

   - `.quick-view form .row .image-container .sub-image img`: Aturan ini mengubah tampilan gambar yang berada di dalam elemen dengan class "sub-image" yang berada di dalam elemen dengan class "image-container" yang berada di dalam elemen dengan class "row" yang berada di dalam elemen form dengan class "quick-view". Aturan ini mengubah lebar gambar menjadi 8rem.
